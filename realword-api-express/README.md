## 项目目录

```
├─app.js
├─package.json
├─util  # 工具模块
├─router  # 路由
├─model  # 数据持久层
├─middleware # 非路由中间件
├─controller  # 用于解析用户的输入，处理后返回相应的结果
├─config  # 配置文件
|   └config.default.js
```
